Project Name: Fitness Tracker

## Description:
A web application created to help users track and improve their exercise habits

## Features:
- User login via Google
- Allows users to schedule exercise sessions
- Provides reminders for scheduled exercises
- Provides relevant statistics on activities and frequency
- Stores user data vs Sqlite3
- Responsive design for mobile and desktop

## Technologies used:
- HTML/CSS
- Javascript
- D3
- Node
- Express
- Sqlite3
- OAuth2
